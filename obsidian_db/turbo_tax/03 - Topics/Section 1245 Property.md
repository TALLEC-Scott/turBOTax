# Section 1245 Property

## Overview

Section 1245 property includes depreciable personal property and certain other property. Upon disposition, gain is recaptured as ordinary income to the extent of depreciation allowed or allowable.

## What Is Section 1245 Property

### Personal Property
- Tangible personal property
- Intangible property (amortizable)

### Other Tangible Property
Property used as:
- Integral part of manufacturing, production, extraction
- Transportation, communications, electricity, gas, water, sewage disposal
- Research facility in above activities
- Bulk storage facility for fungible commodities

### Single Purpose Structures
- Agricultural (livestock) structures
- Horticultural structures

### Storage Facilities
- Petroleum storage facilities
- Primary petroleum product storage

### Other Property
- Railroad grading and tunnel bores
- Qualified production property (after July 4, 2025)

## What Is NOT Section 1245 Property

- Buildings (except qualified production property)
- Structural components of buildings

### Exception
Structures that are essentially machinery or equipment:
- Oil and gas storage tanks
- Grain storage bins
- Silos
- Fractionating towers
- Blast furnaces
- Coke ovens
- Brick kilns
- Coal tipples

## Recapture Rules

### Amount Recaptured
**Lesser of:**
1. Depreciation and amortization allowed or allowable, OR
2. Gain realized (amount realized - adjusted basis)

### Result
- All depreciation recaptured as **ordinary income**
- Any excess gain = [[Section 1231 Gains and Losses|Section 1231 gain]]

## What Counts as Depreciation

| Type | Included |
|------|----------|
| Ordinary depreciation | ✓ |
| Bonus depreciation | ✓ |
| Section 179 deduction | ✓ |
| Amortization (leases, improvements) | ✓ |
| Investment credit basis reduction | ✓ |
| Clean vehicle credit basis reduction | ✓ |
| Reforestation expenditures | ✓ |
| Pollution control facilities | ✓ |

## Example Calculation

Light-duty truck purchased February 2023 for $10,000:
- 2023 depreciation: $2,000
- 2024 depreciation: $3,200
- 2025 depreciation (year of sale): $960
- **Total depreciation: $6,160**

Sold May 2025 for $7,000:
1. Amount realized: $7,000
2. Cost: $10,000
3. Adjusted basis: $10,000 - $6,160 = $3,840
4. Gain realized: $7,000 - $3,840 = $3,160
5. **Ordinary income (lesser of depreciation or gain): $3,160**

## Depreciation Allowed vs. Allowable

- Use the greater of allowed or allowable
- If no depreciation claimed, use straight-line amount
- Applies only for recapture calculation
