# Casualty Loss Deduction Limits

## Overview

After figuring the amount of your casualty or theft loss, deduction limits determine how much of the loss you can actually deduct.

## The Two Limits

### 1. $100 Rule

Reduce each casualty or theft loss by **$100** when figuring your deduction.

| Situation | How Applied |
|-----------|-------------|
| Single event with multiple items | Apply **once** for entire event |
| Multiple events during year | Apply **separately** to each event |
| Multiple owners (not joint filers) | Apply **separately** to each owner |
| Married filing jointly | Apply **once** as if one person |
| Married filing separately | Apply **separately** to each spouse |

**Special Rule**: [[Qualified Disaster Loss|Qualified disaster losses]] use **$500** instead of $100.

### 2. 10% AGI Rule

Reduce total casualty or theft losses from federally declared disasters by **10% of your Adjusted Gross Income (AGI)**.

| Situation | How Applied |
|-----------|-------------|
| Single event | Apply **once** to total loss |
| Multiple federally declared disasters | Apply to **total of all** losses |
| Multiple persons (not joint filers) | Apply **separately** to each person |
| Married filing jointly | Apply **once** as if one person |
| Married filing separately | Apply **separately** to each spouse |

**Special Rule**: [[Qualified Disaster Loss|Qualified disaster losses]] are **NOT subject** to the 10% AGI rule.

## Order of Application

1. First, apply the $100 rule to each separate casualty/theft event
2. Then, apply the 10% AGI rule to total federally declared disaster losses

## Example Calculation

**Regular Federally Declared Disaster Loss:**
```
Loss from tornado damage:        $15,000
Less $100 rule:                   -100
                                  ------
Subtotal:                        $14,900
Less 10% AGI ($60,000 AGI):      -6,000
                                  ------
Deductible loss:                  $8,900
```

**Qualified Disaster Loss:**
```
Loss from qualified disaster:     $15,000
Less $500 rule:                    -500
                                  ------
Deductible loss:                 $14,500
(No 10% AGI reduction)
```

## Personal Casualty Gains Exception

If you have **personal casualty gains** during the year:

1. First offset gains with losses NOT from federally declared disasters
2. Any excess gain reduces losses from federally declared disasters
3. Then apply 10% AGI rule to remaining federal disaster losses

**Example**: $13,000 personal casualty gain, $15,500 non-disaster theft loss, $25,000 federal disaster loss:
- Offset $13,000 gain with $13,000 of theft loss
- Remaining $2,500 theft loss is not deductible (not federal disaster)
- Federal disaster loss of $25,000 remains (subject to $100 and 10% AGI rules)

## Business Property

Losses on **business property and income-producing property** are NOT subject to:
- The $100 rule
- The 10% AGI rule
- The federally declared disaster requirement

These losses are deducted in full (subject to other limitations like passive activity rules).

## Summary Table

| Loss Type | $100 Rule | $500 Rule | 10% AGI Rule |
|-----------|-----------|-----------|--------------|
| Regular casualty (pre-2018 personal) | N/A | N/A | N/A |
| Personal-use (not federal disaster) | N/A | N/A | Not deductible |
| Federal casualty loss | $100 | N/A | Yes |
| Qualified disaster loss | N/A | $500 | No |
| Business/income property | N/A | N/A | No |
