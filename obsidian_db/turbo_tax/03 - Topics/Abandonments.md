# Abandonments

## Overview

Abandonment of property is a disposition that occurs when you voluntarily and permanently give up possession and use of property with the intention of ending ownership, without passing it to anyone else.

## Tax Treatment

### General Rule
- Not treated as a sale or exchange
- Gain if amount realized > adjusted basis
- Loss if adjusted basis > amount realized

### Business or Investment Property
- Abandonment loss is deductible
- Generally treated as **ordinary loss** (not capital)
- Includes leasehold improvements made by lessor

### Personal-Use Property
- **Loss is NOT deductible**
- Example: Abandonment of personal residence

## MACRS Asset Partial Disposition

For abandonment of a portion of a MACRS asset:
- Make partial disposition election
- Report on timely filed return (including extensions)
- Replacement portion classified under same asset class as disposed portion

## Secured Property Rules

### Recourse Debt
- May have ordinary income from canceled debt
- File [[Form 982]] if exclusion applies

### Nonrecourse Debt
- Different rules apply
- See [[Pub 4681 - Canceled Debts, Foreclosures, Repossessions, and Abandonments]]

## Information Returns

- **Form 1099-A**: Lender sends if property is abandoned and lender knows
- **Form 1099-C**: Lender sends if debt is canceled ($600+)
