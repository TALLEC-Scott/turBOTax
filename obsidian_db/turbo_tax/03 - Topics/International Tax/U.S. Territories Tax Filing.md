# U.S. Territories Tax Filing

## Overview

Individuals with income from U.S. territories may need to file tax returns with the territory, the United States, or both. The filing requirements depend on residency status and the source of income.

## Covered Territories

- American Samoa
- Commonwealth of the Northern Mariana Islands (CNMI)
- Guam
- Puerto Rico
- U.S. Virgin Islands (USVI)

## Filing Matrix by Residency Status

### Bona Fide Residents

| Territory | File With | Income Reported |
|-----------|-----------|-----------------|
| American Samoa | AS + U.S. | Worldwide on AS return; exclude AS source on U.S. return (Form 4563) |
| CNMI | CNMI only | Worldwide income |
| Guam | Guam only | Worldwide income |
| Puerto Rico | PR + U.S. (if U.S. income) | Worldwide on PR return; only U.S. source on U.S. return |
| USVI | USVI only | Worldwide income (+ Form 1040INFO for non-USVI source) |

### U.S. Citizens/Residents (Not Bona Fide Residents)

| Territory | File With | Income Reported |
|-----------|-----------|-----------------|
| American Samoa | U.S. + AS | Worldwide on U.S.; AS source only on AS return |
| CNMI | U.S. + Form 5074 | Worldwide (if AGI ≥ $50,000 + $5,000 CNMI income) |
| Guam | U.S. + Form 5074 | Worldwide (if AGI ≥ $50,000 + $5,000 Guam income) |
| Puerto Rico | U.S. | Worldwide; credit for PR taxes paid |
| USVI | U.S. + Form 8689 | Worldwide; allocate tax to USVI |

## Key Forms by Territory

### American Samoa
- **Form 4563**: Attach to U.S. return to exclude American Samoa source income
- File with: American Samoa Government Tax Office, Pago Pago, AS 96799

### CNMI
- **Form 5074**: Allocation of Individual Income Tax to CNMI (if AGI ≥ $50,000 and $5,000+ CNMI income)
- File with: CNMI Division of Revenue and Taxation, Saipan, MP 96950

### Guam
- **Form 5074**: Allocation of Individual Income Tax to Guam (if AGI ≥ $50,000 and $5,000+ Guam income)
- File with: Guam Department of Revenue and Taxation, P.O. Box 23607 GMF, Guam 96921

### Puerto Rico
- File with: Puerto Rico Department of Treasury (Hacienda)
- U.S. return required if U.S. source income

### USVI
- **Form 8689**: Allocation of Individual Income Tax to USVI
- **Form 1040INFO**: For bona fide residents with non-USVI source income
- File with: Virgin Islands Bureau of Internal Revenue, St. Thomas, USVI 00802

## Joint Return Rules

When filing a joint return with a spouse who is a bona fide resident of a territory:

**Rule**: File with the jurisdiction where the spouse with the greater AGI would file separately.

### Example
- Spouse A (U.S. resident): $65,000 income
- Spouse B (CNMI bona fide resident): $20,000 income
- Result: File joint return with United States

## Special Filing Situations

### Self-Employment Tax

Bona fide residents of all territories must pay self-employment tax to the U.S. on net earnings of $400+:
- File **Form 1040-SS** if no U.S. income tax filing requirement
- Attach **Schedule SE** to Form 1040 if U.S. filing required

### Additional Medicare Tax

May apply to:
- Wages, RRTA compensation, and self-employment income above thresholds
- Thresholds: $125,000 (MFS), $250,000 (MFJ), $200,000 (Single, HOH, QSS)

### Net Investment Income Tax (NIIT)

3.8% tax may apply to bona fide residents of **Puerto Rico** and **American Samoa** if:
- Federal filing obligation exists
- Modified AGI from non-territory sources exceeds threshold

**Note**: CNMI, Guam, and USVI bona fide residents generally do not have federal filing obligation, so NIIT generally doesn't apply.

### Estimated Tax Payments

Pay to the jurisdiction where you would file your return:
- First payment determines jurisdiction for the year
- Joint payments: Pay where spouse with greater estimated AGI would file

## Extension of Time to File

### Automatic 2-Month Extension
Available if living/working outside U.S. and Puerto Rico on due date:
- File by June 15
- Interest still accrues from April 15
- Attach statement explaining qualification

### Additional Extensions
- Form 4868: Additional 4 months (to October 15)
- Letter request: Additional 2 months (to December 15)
