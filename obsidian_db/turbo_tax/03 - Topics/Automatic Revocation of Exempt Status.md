# Automatic Revocation of Exempt Status

## Overview

Tax-exempt organizations that fail to file required annual returns or notices for three consecutive years automatically lose their tax-exempt status. This rule applies to most organizations exempt under Section 501(a).

## The Three-Year Rule

### Organizations Affected
All organizations required to file annual returns:
- Form 990 filers
- Form 990-EZ filers
- Form 990-N (e-postcard) filers
- Form 990-PF filers (private foundations)

### What Counts
- Failure to file for three consecutive years
- Includes failure to file any required return
- Late filings count as failures
- Incomplete returns may count as failures

### Effective Date of Revocation
- Effective as of the due date of the third return
- Occurs automatically by operation of law
- No IRS notice required for revocation to take effect

## Notice of Revocation

### IRS Publication
- IRS publishes list of revoked organizations
- Available on IRS.gov
- Organization's name and EIN listed

### Notification
- IRS sends notice to organization's last known address
- Notice explains revocation and reinstatement procedures

## Consequences of Revocation

### Tax Consequences
- Organization is taxable from revocation date
- Must file corporate income tax returns
- May owe income tax, estimated tax penalties

### Deductibility Impact (501(c)(3) Organizations)
- Donor contributions no longer deductible
- Donors who relied in good faith may be protected
- Publication provides constructive notice

### Employment Tax Impact
- May affect payroll processing
- May affect fringe benefit eligibility

## Reinstatement Procedures

### Within 15 Months of Revocation
**Streamlined Process:**
- File Form 1023 or 1024 (as applicable)
- State reason for failure to file
- Certify compliance for current and future years
- Pay reduced user fee
- Exemption reinstated retroactively if approved

### After 15 Months
**Standard Process:**
- File Form 1023 or 1024 with full application
- Request retroactive reinstatement
- Show reasonable cause for failure to file
- May be granted with or without retroactivity

### No Retroactive Reinstatement
If organization cannot show reasonable cause:
- Exemption effective only from date of new application
- Organization taxable for gap period

## Reasonable Cause Factors

Factors considered for retroactive reinstatement:
1. Reasonable cause for failure to file
2. Whether organization actually filed returns
3. Whether organization acted in good faith
4. Small organization status
5. Reliance on tax professional
6. Events beyond organization's control

## Group Exemption Impact

### Central Organization Revocation
If central organization's exemption is revoked:
- All subordinate organizations lose recognized status
- Subordinates must apply individually or with new central organization

### Subordinate Revocation
If only subordinate is revoked:
- Other subordinates unaffected
- Revoked subordinate must apply individually

## Prevention

### Best Practices
- Calendar all filing deadlines
- File extensions if needed
- Review filing requirements annually
- Maintain current address with IRS
- Monitor IRS.gov for status updates

### Missing a Return
If organization misses a return:
- File immediately upon discovery
- Request abatement of penalties if reasonable cause
- Document reason for late filing
- Set up systems to prevent recurrence
