# Classes of Assets

## Overview

When a business is sold, the purchase price must be allocated among the assets using the residual method. Assets are classified into seven classes, and consideration is allocated in a specific order.

## Allocation Order

Consideration is allocated to each class before moving to the next. Amount allocated cannot exceed FMV (except Class VII).

### Class I - Cash and Deposits
- Cash
- General deposit accounts (checking, savings)
- **Excludes**: Certificates of deposit

### Class II - Financial Assets
- Certificates of deposit
- U.S. Government securities
- Foreign currency
- Actively traded personal property
- Stocks and securities

### Class III - Receivables
- Accounts receivable
- Debt instruments
- Other assets marked to market

### Class IV - Inventory
- Stock in trade
- Property held for sale to customers

### Class V - Tangible Business Property
- Office furniture and fixtures
- Equipment
- Buildings (except Class VI buildings)
- Land
- Vehicles

### Class VI - Section 197 Intangibles
- Customer lists
- Patents and copyrights
- Franchises
- Trademarks
- Non-compete agreements
- **Excludes**: Goodwill, going concern value

### Class VII - Goodwill and Going Concern
- Goodwill
- Going concern value
- **Can exceed FMV**

## Residual Method Calculation

1. Start with total consideration
2. Subtract Class I assets (cash received)
3. Allocate remaining to Class II (up to FMV)
4. Continue through Class VII
5. Any remaining amount goes to Class VII

## Example Allocation

| Class | Asset | FMV | Allocated Amount |
|-------|-------|-----|------------------|
| I | Cash | $10,000 | $10,000 |
| II | Marketable securities | $15,000 | $15,000 |
| III | Accounts receivable | $20,000 | $20,000 |
| IV | Inventory | $30,000 | $30,000 |
| V | Equipment | $40,000 | $40,000 |
| VI | Patents | $25,000 | $25,000 |
| VII | Goodwill | ? | $100,000 (residual) |
| **Total** | | | **$240,000** |

## Reporting Requirements

- [[Form 8594]] filed by both buyer and seller
- Due with tax return for year of acquisition
- Attach to buyer's and seller's returns
